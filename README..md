A Java-based **Movie Ticket Booking System** that allows users to browse movies, select seats, and book tickets from the console.

This project demonstrates core **Object-Oriented Programming** concepts including classes, methods, collections, and user input handling.


## Features

- View list of available movies
- Select movie and showtime
- Choose seats for booking
- Confirm ticket bookings
- Display booked reservations
